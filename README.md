# Pre-Proyecto-PymeSync
Este proyecto trata sobre crear un sistema pensado para ayudar a pequeñas y medianas empresas (PYMES) a organizar mejor su trabajo.  Hoy en día, muchos negocios manejan sus ventas, productos y clientes de forma desordenada o usando varias herramientas distintas. La idea de este sistema es juntar todo en un solo lugar para que sea más fácil de usar.
