# Pre-Proyecto-PymeSync

Este proyecto trata sobre crear un sistema pensado para ayudar a pequeñas y medianas empresas (PYMES) a organizar mejor su trabajo. 

## 🎯 Objetivo
Juntar en un solo lugar la gestión de:
* Ventas y Facturación.
* Control de Inventario (productos).
* Base de datos de Clientes.
